APIs

A couple APIs I host here.

Artwork

Fetches album artwork from last.fm (highest quality available)
Usage:

  • ingvaras.com/artwork/Blur/The+Magic+Whip returns HTML
  • ingvaras.com/artwork/Blur/The+Magic+Whip/raw returns PNG/JPEG

Text to speech (offline)

(Ab)uses Google Translate. Can be used

  • ingvaras.com/tts/fr/Elon+Musk+est+génial returns MP3 audio
